Detailed Description

Data structure for programming motor movement using S_JOG

Parameters
[in]pIDid of the motor
[in]pos(1) position of the motor when mode = 0 (0 to 1023). (2) speed of the motor when mode = 1 (0 to 1023)
[in]timeoperating time to reach position or continue speed (0 to 255 ticks). every tick = 11.2ms
[in]ledled colour (green = 0x01, blue = 0x02, red = 0x04)
[in]modethe control mode (position control = 0, continuous rotation = 1)
